Contributors: Name

Requires at least: 5.8

Create multi-level category menus with 5-level depth.

== Description ==
Features:
- 5-level category menus
- Gutenberg block support (in progress)
- Widget support
- Shortcode [mlcm_menu]
- Customizable labels
- Responsive design
- Cache system
- AJAX loading
- Deletion in slug ‘category’


== Installation ==
1. Upload the plugin files to /wp-content/plugins/
2. Activate through 'Plugins' menu
3. Configure in Settings → Category Menu
